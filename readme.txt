== Description ==

Cool Kids Network is a WordPress user management system with role-based access.

== Features ==

-   Create users with different roles
-   Manage users with different roles
-   Delete users
-   Edit users
-   Reset users password

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Access the plugin functionality through the "Cool Kids Network" section in your WordPress admin dashboard.
